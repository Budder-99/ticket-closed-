# Autotask Confetti

Autotask Confetti adds a celebratory burst of confetti when you close a ticket in Autotask. 
It includes a popup so you can tweak how the effect looks and feels without touching the code.

## Settings

- **Confetti pieces**: Controls how many confetti pieces appear per burst.
- **Confetti size**: Adjusts how large each piece is.
- **Velocity / spread**: Changes how far and fast the confetti flies.
- **Duration**: Sets how long the animation lasts.
- **Colour theme**: Choose between Classic, Pink pastel, and Company colours.
- **Origin**: Pick whether the burst starts from the top corners or bottom corners.
- **Physics mode**: Toggles the smoother physics-based animation.

## Loading the extension

1. Open `chrome://extensions`.
2. Turn on **Developer mode**.
3. Click **Load unpacked**.
4. Select this project folder.
5. Open the popup to adjust settings and click **Save settings**.

## Using it

- Open an Autotask ticket.
- Close it with **Save & Close** to trigger confetti automatically.
- Use **Test confetti now** in the popup to try it manually.
